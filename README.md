# annot2md
KoBo .annot file to .md conversion

Based on code in https://www.edwinwenink.xyz/posts/33-epub_annotation_export/

---

Possible inputs:
- fzf file selection (file list with 1 file / line)
- single file path

Outputs:
converted file in .md format
